### Arvore oportunidade (Teresa Torres)
Estrutura visual que conecta outcomes a oportunidades e soluções:

```
Outcome desejado (ex: reduzir tempo de relatório de 2h para 20min)
  └── Oportunidade 1: usuário perde tempo buscando dados em sistemas diferentes
        └── Solução A: painel unificado de dados
              └── Experimento: protótipo com 5 usuários em 2 semanas
        └── Solução B: integração automática via API
              └── Experimento: spike técnico de 3 dias
  └── Oportunidade 2: usuário não sabe quais dados incluir
        └── Solução: template guiado com exemplos
```

Use a arvore de oportunidade para organizar o espaço de oportunidades antes de comprometer com soluções.
